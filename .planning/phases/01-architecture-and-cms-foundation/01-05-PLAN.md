   - When `schemaEnabled` is true (default true), inject FAQPage JSON-LD:
     ```typescript
     const faqSchema = {
       '@context': 'https://schema.org',
       '@type': 'FAQPage',
       mainEntity: questions.map((q) => ({
         '@type': 'Question',
         name: q.question,
         acceptedAnswer: {
           '@type': 'Answer',
           text: q.answer,
         },
       })),
     }
     ```
   - Inject via `<script type="application/ld+json" dangerouslySetInnerHTML={{ __html: JSON.stringify(faqSchema).replace(/</g, '\\u003c') }} />`
   - IMPORTANT: The Accordion component from shadcn/ui uses Radix, which requires client-side interactivity. If the generated Accordion component is a Client Component, that is fine -- the FAQBlock itself can still be a Server Component that imports it.
